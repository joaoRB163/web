Formas de utilizar CSS em documentos HTML

CSS inline

É possível utilizar CSS diretametne nas tags do HTML utilizando o atributo "style".

Esse é um exemplo de CSS inline

        <p style="color: red; background-color: lightblue; padding: 10px;">Esse é um exemplo de CSS inline</p>

CSS utilizando a tag style

É possível utilizar a tag <style> para aplicar comandos de estilo via CSS

Esse é um exemplo de CSS aplicado com a tag style 
        <style>
        span{
            color: darkviolet;
            background-color: chartreuse;
            padding: 10px;
        }
        </style>

Folha de estilo fora do documento HTML

É possível importar uma folha de estilo CSS sem a necessidade de escrever dentro do documento HTML.

        <link rel="stylesheet" href="style.css">

Estilização por ID e CLASSE

Esse é um exemplo de estilização via ID

        <p id="id1">Esse é um exemplo de estilização via ID</p>

Esse é um exemplo de estilização via classe

        <p class="class1">Esse é um exemplo de estilização via classe</p>
