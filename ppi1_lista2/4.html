A regra li{color:red;} define a cor VERMELHA para o texto de todos os tópicos que serão utilizados no documetno HTML

A regra .churras{color:green;} vai definir a cor VERDE para todas as tags que utilizarem a classe churras

A regra .fruta{color:gray} vai definir a cor CINZA para todas as tags que utilizarem a classe fruta

A regra li+li.churras{color: black;} vai definir a cor PRETA para todos os elementos li irmãos que utilizarem a classe churras

A regra .churras[draggable="true"]{color: magenta;} vai definir a cor MAGENTA para toda tag que utilizar a classe churras e tiver a propriedade draggable="true"

A regra #lip1,#lip2,#lip3{color:blue} vai aplicar a cor AZUL para as tags que utilizarem os IDs #lip1, #lip2, ou #lip3

A regra #lip3.fruta{color:orange} vai definir a cor LARANJA para tag que utilizar o ID #lip3 e a classe fruta

A regra ol>li.churras#lip5{color:purple} irá definir a cor ROXA para tag li que for filha de uma tag ol e utilizar o ID #lip5 e a classe churras
