O que é CSS?

CSS é a linguagem de estilização e seu propósito é definir como os elementos HTML selecionados devem ser estilizados, controlando sua aparência, layout e comportamento visual.

Exemplo de estilização via CSS

Esse é um exemplo de estilização CSS 

        span{
            background-color: lightblue;
            text-decoration: underline;
            font-family: Arial, Helvetica, sans-serif;
            font-weight: bold;
            padding: 10px;
        }
