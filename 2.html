O que é uma regra CSS

A regra CSS é a regra que aplicará estilo a página HTML atráves de seletores, propriedades e valores indicados

  • Seletor: é aquele que irá apontar em qual lugar o estilo deve ser aplicado
  • Propriedade: é a que "seleciona" o tipo do estilo que será formatado
  • Valores: os valores estão ligados com as propriedades e definem quanto ou como o estilo será configurado
        #exemplo { /* #exemplo é o nosso seletor */
            background-color: blue; /* background-color, color, font-weight e text-decorarion são nossas propriedades */
            color: white; 
            font-weight: bold;
            text-decoration: underline; /* blue, white, bold e underline são os valores que as propriedades devem aplicar no seletor */
        }

Esse é um exemplo de regra CSS

  1. Pseudoclasse

    2. Dentro das regras CSS é possível realizar seleções mais específicas através das Pseudoclasses.

    Em um conjunto de elementos, é possível realizar a seleção apenas do primeiro ou do último elemento conforme o exemplo a seguir:

            .box:first-child{
            background-color: blueviolet;
        }
        .box:last-child{
            background-color: darkgreen;
        }
