O que é HTML?

HTML é a uma linguagem de marcação feita para estruturar elementos em arquivos de texto que ficarão armazenados páginas na WEB.

    Estrutura mínima que é necessário para um arquivo HTML5

  • <html>
  • <head>
  • <body>

Formatação básica

<html>
    <head>
    </head>

    <body>
    </body>
</html>

Formatação completa

<html>
    <!DOCTYPE html> <!-- Indica no documento qual o tipo dele -->
    <head>
        <meta charset="UTF-8"> <!-- Indica qual padrão alfanumérico está sendo utilizado -->
        <title>Título</title> <!-- Configura o nome que aparecerá na aba do navegador -->
    </head>

    <body>
    </body>
</html>
