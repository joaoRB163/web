O que é JavaScript?

O JavaScript foi criado para tornar as páginas web mais dinâmicas e interativas no navegador, aliviando o processamento do servidor ao realizar validações e manipulações simples de páginas WEB. Essa execução no navegador ocorre em um ambiente fechado chamado de sandbox, que impede que o código acesse recursos do sistema operacional do usuário por motivos de segurança.
Para interagir com a página, são fornecidos objetos globais para manipulação de uma página, como: document, consoles e alert
